### duration

不透明度　あいまいさ

### scope

## ajax = Asynchronous JavaScript and XML
<b>「（ページを動かさず）裏側でこっそり通信して、画面を書き換えるJS術」</b>
<br>Asynchronous　非同期
<br>JavaScript　命令を出す実行役
<br>and XML　データの形式（昔の主流 今はajaxが主流）
### asynchronous
非同期性 非同時性
### synchronous
同期生 同時性
## JSON = JavaScript Object Notation
<b>JavaScriptのオブジェクトの書き方</b>
### object
物体 対象 情報の塊
### notation
表記法 記号法 表示 メモ 記録
